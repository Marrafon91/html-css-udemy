Tipografia

Para definir tipografia em HTML e CSS, utilize a propriedade font-family no CSS para escolher a família de fontes, a propriedade font-size para o tamanho da fonte, font-weight para o peso (negrito, por exemplo) e font-style para itálico. Também é possível carregar fontes personalizadas utilizando a regra @font-face.
